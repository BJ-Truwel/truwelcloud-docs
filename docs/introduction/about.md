`TruwelCloud` 是一套面向多源异构数据（结构化数据、图像、视频）的 `SaaS` 级解决方案。系统实现了业务数据全流程的集约化治理，涵盖数据采集接入、弹性传输存储、容灾备份及深度分析展示，确保业务的高效流转与数据安全。

- <Badge text="支持 MQTT 消息接入"/>

- <Badge text="支持数据报表一键导出"/>

- <Badge text="支持物候照片及数据文件管理"/>

- <Badge text="应用接口符合 RESTful API 规范"/>
- <Badge text="支持符合国标 GB28181 监控视频接入"/>
- <Badge text="可部署在用户 Windows / Linux 服务器"/>

- JRE >= 1.8
- MySQL >= 5.7
- Redis >= 7.0.0
- Kafka = 2.8.1
- Zookeeper = 3.5.9
- MQTT Broker = 4.3.12.7
- 时序数据库 >= 3.2.0.0
- 8GiB内存
- 5Mbps带宽  
- 4vCPU(2.55GHz 3.5 GHz)
- 100GiB(ESSD PL0硬盘)
- Windows Server or Centos7

## 浏览器支持

::: warning 
不支持`IE`
:::
